Common Problem Patterns

Two Sum

Problem: Find two numbers that add up to a target.

Easy 
def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

# Example: two_sum([2, 7, 11, 15], 9) -> [0, 1]

Remove Duplicates from Sorted Array

Problem: Remove duplicates in-place and return new length.

Easy 
def removeDuplicates(nums):
    if not nums:
        return 0
    
    i = 0
    for j in range(1, len(nums)):
        if nums[j] != nums[i]:
            i += 1
            nums[i] = nums[j]
    
    return i + 1

# Example: removeDuplicates([1,1,2]) -> 2

Reverse a String

Problem: Reverse a string in-place.

Easy 
def reverse_string(s):
    left, right = 0, len(s) - 1
    while left < right:
        s[left], s[right] = s[right], s[left]
        left += 1
        right -= 1
    return s

# Example: reverse_string(['h','e','l','l','o']) -> ['o','l','l','e','h']

Medium Problems

Longest Substring Without Repeating Characters

Problem: Find length of longest substring without repeating chars.

Medium 
def lengthOfLongestSubstring(s):
    char_index = {}
    max_length = 0
    start = 0
    
    for end, char in enumerate(s):
        if char in char_index and char_index[char] >= start:
            start = char_index[char] + 1
        
        char_index[char] = end
        max_length = max(max_length, end - start + 1)
    
    return max_length

# Example: lengthOfLongestSubstring("abcabcbb") -> 3

Container With Most Water

Problem: Find maximum area of water container.

Medium 
def maxArea(height):
    left, right = 0, len(height) - 1
    max_area = 0
    
    while left < right:
        width = right - left
        h = min(height[left], height[right])
        area = width * h
        max_area = max(max_area, area)
        
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    
    return max_area

Word Search

Problem: Search for word in 2D board using DFS.

Medium 
def exist(board, word):
    if not board or not word:
        return False
    
    def dfs(i, j, k):
        if k == len(word):
            return True
        if i < 0 or i >= len(board) or j < 0 or j >= len(board[0]):
            return False
        if board[i][j] != word[k]:
            return False
        
        board[i][j] = '#'  # Mark as visited
        
        result = (dfs(i+1, j, k+1) or dfs(i-1, j, k+1) or
                 dfs(i, j+1, k+1) or dfs(i, j-1, k+1))
        
        board[i][j] = word[k]  # Restore
        return result
    
    for i in range(len(board)):
        for j in range(len(board[0])):
            if dfs(i, j, 0):
                return True
    return False

Hard Problems

Median of Two Sorted Arrays

Problem: Find median of two sorted arrays in O(log(min(m,n))).

Hard 
def findMedianSortedArrays(nums1, nums2):
    if len(nums1) > len(nums2):
        return findMedianSortedArrays(nums2, nums1)
    
    m, n = len(nums1), len(nums2)
    low, high = 0, m
    
    while low <= high:
        partition1 = (low + high) // 2
        partition2 = (m + n + 1) // 2 - partition1
        
        left1 = float('-inf') if partition1 == 0 else nums1[partition1 - 1]
        left2 = float('-inf') if partition2 == 0 else nums2[partition2 - 1]
        right1 = float('inf') if partition1 == m else nums1[partition1]
        right2 = float('inf') if partition2 == n else nums2[partition2]
        
        if left1 <= right2 and left2 <= right1:
            if (m + n) % 2 == 0:
                return (max(left1, left2) + min(right1, right2)) / 2
            else:
                return max(left1, left2)
        elif left1 > right2:
            high = partition1 - 1
        else:
            low = partition1 + 1

Trapping Rain Water

Problem: Calculate trapped water between elevations.

Hard 
def trap(height):
    if not height:
        return 0
    
    left, right = 0, len(height) - 1
    left_max, right_max = 0, 0
    water = 0
    
    while left < right:
        if height[left] < height[right]:
            if height[left] >= left_max:
                left_max = height[left]
            else:
                water += left_max - height[left]
            left += 1
        else:
            if height[right] >= right_max:
                right_max = height[right]
            else:
                water += right_max - height[right]
            right -= 1
    
    return water

Problem Solving Tips

  • Understand the problem: Read carefully, identify constraints
  • Plan the approach: Discuss solution before coding
  • Start simple: Brute force first, then optimize
  • Use data structures wisely: Hash maps for O(1) lookup, heaps for ordering
  • Two pointers: Great for sorted arrays and string problems
  • Sliding window: Perfect for subarray/substring problems
  • DFS/BFS: Essential for tree and graph traversals
  • Dynamic Programming: Overlapping subproblems? Use DP
  • Test edge cases: Empty inputs, single elements, duplicates
  • Optimize space: Can you reduce extra space usage?
