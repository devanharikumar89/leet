🧩 LeetCode Patterns

Reusable templates for the most common interview problem patterns.

1. Two Pointers

Use when: a sorted array / linked list, pair-sum, palindrome check, removing duplicates, partitioning.

Time: O(n)  |  Space: O(1)

Template — opposite ends 
def two_sum_sorted(arr, target):
    l, r = 0, len(arr) - 1
    while l < r:
        s = arr[l] + arr[r]
        if s == target: return [l, r]
        if s < target:  l += 1
        else:           r -= 1
    return [-1, -1]
Template — same direction (fast/slow) 
def remove_duplicates(arr):
    if not arr: return 0
    slow = 0
    for fast in range(1, len(arr)):
        if arr[fast] != arr[slow]:
            slow += 1
            arr[slow] = arr[fast]
    return slow + 1

2. Sliding Window

Use when: longest/shortest/contains subarray or substring satisfying a condition.

Time: O(n)  |  Space: O(k) for the window state.

Template — variable-size window 
from collections import Counter

def longest_substring_k_distinct(s, k):
    counts = Counter()
    l = best = 0
    for r, ch in enumerate(s):
        counts[ch] += 1
        while len(counts) > k:
            counts[s[l]] -= 1
            if counts[s[l]] == 0:
                del counts[s[l]]      # keep window state clean
            l += 1
        best = max(best, r - l + 1)
    return best
Template — fixed-size window 
def max_sum_window(arr, k):
    window = sum(arr[:k])
    best = window
    for i in range(k, len(arr)):
        window += arr[i] - arr[i - k]
        best = max(best, window)
    return best

3. Fast & Slow Pointers (Floyd's Tortoise & Hare)

Use when: cycle detection, finding the middle of a linked list, "happy number"-style state machines.

Time: O(n)  |  Space: O(1)

Template — cycle detection & cycle start 
def detect_cycle_start(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            # phase 2: walk one from head, one from meeting point
            p = head
            while p is not slow:
                p, slow = p.next, slow.next
            return p
    return None

4. Prefix Sum / Difference Array

Use when: repeated range-sum queries, subarray-sum-equals-k, range increments.

Time: build O(n), query O(1)  |  Space: O(n)

Template — subarray sum equals k (hash map of prefix sums) 
from collections import defaultdict

def subarray_sum_equals_k(nums, k):
    seen = defaultdict(int)
    seen[0] = 1            # empty prefix
    cur = ans = 0
    for x in nums:
        cur += x
        ans += seen[cur - k]
        seen[cur] += 1
    return ans

5. Binary Search

Use when: sorted input, or any monotonic predicate f(x) -> bool — "smallest x that works".

Time: O(log n) over the search space.

Template — classic find 
def bsearch(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target: return mid
        if arr[mid] < target:  lo = mid + 1
        else:                  hi = mid - 1
    return -1
Template — "binary search on the answer" (lower bound) 
def smallest_x_such_that(feasible, lo, hi):
    # find min x in [lo, hi] with feasible(x) == True
    while lo < hi:
        mid = (lo + hi) // 2
        if feasible(mid): hi = mid
        else:             lo = mid + 1
    return lo

# bisect module: standard library
import bisect
i = bisect.bisect_left(arr, target)   # leftmost insertion point
j = bisect.bisect_right(arr, target)  # rightmost insertion point

6. BFS (Breadth-First Search)

Use when: shortest path in an unweighted graph/grid, level-order tree traversal, "minimum steps".

Time: O(V + E)  |  Space: O(V)

Template — grid BFS 
from collections import deque

DIRS = [(1,0), (-1,0), (0,1), (0,-1)]

def bfs(grid, start):
    R, C = len(grid), len(grid[0])
    q = deque([(start, 0)])
    seen = {start}
    while q:
        (r, c), d = q.popleft()
        # ... process (r, c) at distance d ...
        for dr, dc in DIRS:
            nr, nc = r + dr, c + dc
            if 0 <= nr < R and 0 <= nc < C and (nr, nc) not in seen and grid[nr][nc] != '#':
                seen.add((nr, nc))
                q.append(((nr, nc), d + 1))

7. DFS / Backtracking

Use when: all paths/permutations/combinations/subsets, constraint satisfaction (Sudoku, N-queens), tree/graph traversal.

Time: often exponential; prune aggressively.

Template — backtracking (subsets / permutations / combinations) 
def subsets(nums):
    res, path = [], []
    def dfs(i):
        if i == len(nums):
            res.append(path.copy())   # ⚠ copy! path is mutated
            return
        # exclude
        dfs(i + 1)
        # include
        path.append(nums[i])
        dfs(i + 1)
        path.pop()
    dfs(0)
    return res
Template — recursive DFS on a graph 
def dfs(node, graph, seen):
    if node in seen: return
    seen.add(node)
    for nb in graph[node]:
        dfs(nb, graph, seen)

8. Heap / Top-K

Use when: top-K, K-th smallest/largest, merge K sorted lists, scheduling, Dijkstra.

Time: O(n log k) for top-K; O((V+E) log V) for Dijkstra.

Template — top-K largest with a min-heap of size k 
import heapq

def top_k(nums, k):
    h = []                              # min-heap
    for x in nums:
        heapq.heappush(h, x)
        if len(h) > k:
            heapq.heappop(h)            # drop smallest -> keep k largest
    return h                             # the k largest, unordered

# Max-heap trick: push negated values
heapq.heappush(h, -x); -heapq.heappop(h)

# Dijkstra (sparse, non-negative weights)
def dijkstra(graph, src):
    dist = {src: 0}
    pq = [(0, src)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]: continue
        for v, w in graph[u]:
            nd = d + w
            if nd < dist.get(v, float('inf')):
                dist[v] = nd
                heapq.heappush(pq, (nd, v))
    return dist

9. Hash Map / Set (counting, lookups, dedup)

Use when: O(1) lookups by key, frequency counting, anagram grouping, "seen before".
Template — Two Sum (one pass) 
def two_sum(nums, target):
    seen = {}                         # value -> index
    for i, x in enumerate(nums):
        if target - x in seen:
            return [seen[target - x], i]
        seen[x] = i
Template — group anagrams 
from collections import defaultdict

def group_anagrams(words):
    groups = defaultdict(list)
    for w in words:
        key = ''.join(sorted(w))      # or tuple of letter counts
        groups[key].append(w)
    return list(groups.values())

10. Monotonic Stack / Queue

Use when: "next greater/smaller element", largest rectangle in histogram, sliding-window maximum.

Time: O(n) amortized.

Template — next greater element 
def next_greater(nums):
    res = [-1] * len(nums)
    stack = []                        # indices, values strictly decreasing
    for i, x in enumerate(nums):
        while stack and nums[stack[-1]] < x:
            res[stack.pop()] = x
        stack.append(i)
    return res
Template — sliding-window max (monotonic deque) 
from collections import deque

def sliding_max(nums, k):
    dq = deque()                      # store indices, values decreasing
    out = []
    for i, x in enumerate(nums):
        while dq and dq[0] <= i - k:        # drop out-of-window
            dq.popleft()
        while dq and nums[dq[-1]] < x:      # maintain monotonicity
            dq.pop()
        dq.append(i)
        if i >= k - 1:
            out.append(nums[dq[0]])
    return out

11. Dynamic Programming

Use when: optimal substructure + overlapping subproblems — counting, optimization, decision over sequences/grids.
Template — top-down with memoization 
from functools import lru_cache

def coin_change(coins, amount):
    @lru_cache(maxsize=None)
    def dp(rem):
        if rem == 0: return 0
        if rem < 0:  return float('inf')
        return 1 + min(dp(rem - c) for c in coins)
    ans = dp(amount)
    return ans if ans != float('inf') else -1
Template — bottom-up (tabulation) 
def coin_change(coins, amount):
    INF = float('inf')
    dp = [0] + [INF] * amount
    for a in range(1, amount + 1):
        for c in coins:
            if c <= a:
                dp[a] = min(dp[a], dp[a - c] + 1)
    return dp[amount] if dp[amount] != INF else -1
Template — 2D grid DP (unique paths) 
def unique_paths(m, n):
    dp = [[1] * n for _ in range(m)]
    for r in range(1, m):
        for c in range(1, n):
            dp[r][c] = dp[r-1][c] + dp[r][c-1]
    return dp[m-1][n-1]

12. Greedy

Use when: a local optimum provably leads to a global optimum — interval scheduling, jump game, Huffman.
Template — interval scheduling (max non-overlap) 
def max_non_overlapping(intervals):
    intervals.sort(key=lambda x: x[1])      # sort by END time
    count, end = 0, float('-inf')
    for s, e in intervals:
        if s >= end:
            count += 1
            end = e
    return count

13. Union-Find (Disjoint Set Union)

Use when: connectivity queries, counting connected components, Kruskal's MST, cycle detection in an undirected graph.

Time: ~O(α(n)) per op (essentially constant).

Template — path compression + union by rank 
class DSU:
    def __init__(self, n):
        self.p = list(range(n))
        self.r = [0] * n
    def find(self, x):
        while self.p[x] != x:
            self.p[x] = self.p[self.p[x]]   # path compression
            x = self.p[x]
        return x
    def union(self, a, b):
        ra, rb = self.find(a), self.find(b)
        if ra == rb: return False
        if self.r[ra] < self.r[rb]: ra, rb = rb, ra
        self.p[rb] = ra
        if self.r[ra] == self.r[rb]: self.r[ra] += 1
        return True

14. Topological Sort (Kahn's Algorithm)

Use when: course schedule, build order, any DAG ordering with dependencies.

Time: O(V + E)

Template — BFS / Kahn 
from collections import defaultdict, deque

def topo_sort(n, edges):                   # edges: (u -> v)
    g = defaultdict(list)
    indeg = [0] * n
    for u, v in edges:
        g[u].append(v); indeg[v] += 1
    q = deque(i for i in range(n) if indeg[i] == 0)
    order = []
    while q:
        u = q.popleft(); order.append(u)
        for v in g[u]:
            indeg[v] -= 1
            if indeg[v] == 0: q.append(v)
    return order if len(order) == n else []   # [] -> cycle exists

15. Tree Traversals

Use when: any binary-tree problem — recursion is usually cleanest.
Templates — DFS variants & BFS level-order 
# Recursive DFS
def inorder(node, out):
    if not node: return
    inorder(node.left, out)
    out.append(node.val)
    inorder(node.right, out)

# Iterative inorder with an explicit stack
def inorder_iter(root):
    stack, out, cur = [], [], root
    while cur or stack:
        while cur:
            stack.append(cur); cur = cur.left
        cur = stack.pop()
        out.append(cur.val)
        cur = cur.right
    return out

# BFS level order
from collections import deque
def level_order(root):
    if not root: return []
    q, out = deque([root]), []
    while q:
        level = []
        for _ in range(len(q)):
            n = q.popleft(); level.append(n.val)
            if n.left:  q.append(n.left)
            if n.right: q.append(n.right)
        out.append(level)
    return out

16. Trie (Prefix Tree)

Use when: prefix queries, autocomplete, word-search-II style problems.
Template — minimal dict-based trie 
class Trie:
    def __init__(self):
        self.root = {}
    def insert(self, word):
        node = self.root
        for ch in word:
            node = node.setdefault(ch, {})
        node['$'] = True               # end-of-word marker
    def search(self, word):
        node = self.root
        for ch in word:
            if ch not in node: return False
            node = node[ch]
        return '$' in node
    def startsWith(self, prefix):
        node = self.root
        for ch in prefix:
            if ch not in node: return False
            node = node[ch]
        return True

17. Bit Manipulation

Use when: subsets via bitmask, single-number XOR tricks, power-of-two checks, DP over subsets.
Common tricks 
x & (x - 1)              # clears lowest set bit;  == 0 iff x is power of two
x & -x                   # isolates lowest set bit
bin(x).count('1')        # popcount
x ^ y                    # toggle / find unique element (XOR all)
x | (1 << i)             # set bit i
x & ~(1 << i)            # clear bit i
(x >> i) & 1             # read bit i

# Iterate over all subsets of a bitmask m
sub = m
while sub:
    # ... use sub ...
    sub = (sub - 1) & m
# (the empty subset 0 is handled separately if needed)

Pattern Picker (Quick Heuristics)

  • "Sorted array + pair/triplet" → Two Pointers
  • "Longest/shortest contiguous ... with constraint" → Sliding Window
  • "Cycle / middle of linked list" → Fast & Slow
  • "Sum of range / subarray sum = k" → Prefix Sum
  • "Sorted input OR monotonic predicate" → Binary Search
  • "Shortest path in unweighted graph / level-order" → BFS
  • "All combinations / permutations / paths" → DFS + Backtracking
  • "Top-K / K-th / merge K sorted" → Heap
  • "Count / lookup / dedup by value" → Hash map / set
  • "Next greater / sliding max" → Monotonic Stack/Deque
  • "Optimal value over sequence/grid + overlapping subproblems" → DP
  • "Local choice = global optimum" → Greedy
  • "Connectivity / components" → Union-Find
  • "Dependency ordering on a DAG" → Topological Sort
  • "Prefix queries / autocomplete" → Trie
  • "Subsets up to ~20 elements / XOR puzzles" → Bitmask
