# Python Knowledge Base

A comprehensive repository for learning and mastering Python programming, data structures, algorithms, and design patterns.

## Overview

This repository serves as a personal knowledge base for:

- **Python Fundamentals** - Core language features, best practices, and idioms
- **Data Structures** - Arrays, linked lists, trees, graphs, heaps, hash tables, etc.
- **Algorithms** - Sorting, searching, dynamic programming, greedy algorithms, and more
- **Design Patterns** - Architectural patterns, creational, structural, and behavioral patterns
- **Problem Solving** - LeetCode-style problems and real-world coding challenges
- **Code Optimization** - Performance tips, time/space complexity analysis

## 📂 Repository Structure

All content is provided in **HTML format** for easy readability and navigation:

- **index.html** - Main hub with links to all sections
- **python-basics.html** - Data types, control flow, functions, OOP basics
- **data-structures.html** - Implementation and complexity analysis of common data structures
- **algorithms.html** - Sorting, searching, graph algorithms, and dynamic programming
- **design-patterns.html** - Creational, structural, and behavioral design patterns
- **problem-solving.html** - Common coding problems with solutions and explanations
- **cheatsheet.html** - Quick Python reference guide

## 🚀 Getting Started

1. **Clone the repository:**
```bash
git clone https://github.com/devanharikumar89/leet.git
cd leet
```

2. **Open the knowledge base:**
   - Open `index.html` in a web browser to access the interactive wiki
   - Click through sections to explore topics
   - Each page has syntax-highlighted code examples and explanations
